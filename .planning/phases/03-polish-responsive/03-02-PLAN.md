Add new section at end of file:
```css
/* =================================
   14. VISUAL EFFECTS
   ================================= */

/* ---------------------------------
   Grain Texture Overlay
   --------------------------------- */

.grain-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  pointer-events: none;
  z-index: 9999;
  opacity: var(--grain-opacity);
  mix-blend-mode: overlay;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.65' numOctaves='3' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)'/%3E%3C/svg%3E");
}

/* Reduce grain opacity on mobile for performance */
@media (max-width: 767px) {
  :root {
    --grain-opacity: 0.05;
  }
}
```

Key implementation notes:
- `pointer-events: none` is CRITICAL - allows clicking through the overlay
- `z-index: 9999` ensures overlay is above all content
- `mix-blend-mode: overlay` blends grain with background for subtle effect
- SVG feTurbulence is resolution-independent, no image files needed
- Opacity reduced on mobile for performance
